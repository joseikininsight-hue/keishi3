# 用語集ページ（/glossary/）セットアップ手順

## 📋 概要

補助金・助成金用語集ページを作成しました。100語以上の専門用語を収録し、SEO最適化、リアルタイム検索、五十音ナビゲーションを実装しています。

**ページURL:** `https://joseikin-insight.com/glossary/`

---

## 🚀 WordPress管理画面でのページ作成手順

### 1. 固定ページを新規作成

1. WordPress管理画面にログイン
2. **固定ページ > 新規追加** をクリック

### 2. ページ設定

#### 基本情報
- **タイトル:** `用語集` または `補助金・助成金用語集`
- **パーマリンク（URL）:** `glossary`
  - URLが `https://joseikin-insight.com/glossary/` になることを確認

#### テンプレート選択
- 右サイドバーの「ページ属性」セクションを開く
- **テンプレート** ドロップダウンから以下を選択：
  ```
  Glossary Page (補助金・助成金用語集)
  ```

#### その他の設定
- **本文エディタ:** 空白のままでOK（テンプレートが全て処理します）
- **アイキャッチ画像:** 任意（設定しなくてもOK）
- **公開状態:** 「公開」を選択

### 3. 公開

「公開」ボタンをクリックしてページを公開します。

## 🔧 カスタマイズ方法

### 用語の追加
`pages/templates/page-glossary.php` の各セクションに以下の形式で追加：

```php
<article class="glossary-item">
    <div class="glossary-term-header">
        <h3 class="glossary-term">用語名</h3>
        <span class="glossary-reading">（読み仮名）</span>
    </div>
    <p class="glossary-definition">
        用語の説明文をここに記載。
    </p>
</article>
```

### スタイルのカスタマイズ
`<style>` タグ内のCSS変数を変更：

```css
/* カラー調整 */
--primary-color: #FFD500;  /* アクセントカラー */
--text-color: #1a1a1a;     /* テキストカラー */
--border-color: #e0e0e0;   /* ボーダーカラー */
```

## 🆘 トラブルシューティング

### ページが表示されない
1. パーマリンク設定を確認
2. テンプレートが正しく選択されているか確認
3. ファイルのアップロードを確認

### 検索機能が動作しない
1. JavaScriptエラーがないか確認（ブラウザのコンソール）
2. キャッシュをクリア

### スタイルが崩れる
1. テーマのCSSとの競合を確認
2. ブラウザキャッシュをクリア
3. 別のブラウザで確認
